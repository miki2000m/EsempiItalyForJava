SCALETTA DI STUDIO COMPLETA - PROGETTO JAVAdallaBasi

Questo file è la tua roadmap per studiare Java partendo da zero fino a raggiungere un livello avanzato, usando i package di questo progetto.
Ogni argomento è pensato per costruire le basi per quello successivo.

--- METODO DI STUDIO CONSIGLIATO PER OGNI PACKAGE ---

Per ogni package elencato di seguito, segui questi 4 passaggi:
1.  **Leggi la Teoria:** Apri e leggi attentamente il file `Riepilogo_*.txt` per capire i concetti teorici.
2.  **Analizza la Pratica:** Apri il file `Main*.java` e leggi il codice e i commenti per vedere come la teoria viene applicata.
3.  **Esegui il Codice:** Fai partire il `main` e osserva l'output sulla console. Assicurati di capire cosa sta succedendo.
4.  **Sperimenta:** Questo è il passo più importante. Modifica il codice. Cambia i valori. Prova a "romperlo" e vedi che errori ottieni.
        Fatti la domanda: "Cosa succede se...?" e trova la risposta.

MODULO 1: LE BASI FONDAMENTALI DEL LINGUAGGIO
==================================================
Obiettivo: Scrivere i primi, semplici programmi.

1.  **Variabili e Tipi di Dato:**
    *   PACKAGE: `MODULO_1_BASI_E_CONTROLLO/Variabili`
    *   PACKAGE: `_strumenti_di_studio/TipiPrimitivi`
    *   Impara la differenza tra tipi primitivi (int, double) e tipi riferimento (String), e a cosa servono le classi Wrapper. Scopri la keyword `var`.

2.  **Controllo del Flusso:**
    *   PACKAGE: `MODULO_1_BASI_E_CONTROLLO/Switch_If_for`
    *   Impara a controllare il tuo programma con `if/else`, `switch` e i vari tipi di cicli (`for`, `while`).

3.  **Metodi:**
    *   PACKAGE: `MODULO_1_BASI_E_CONTROLLO/Methods`
    *   Impara a organizzare il codice in blocchi riutilizzabili (i metodi) e a usare i `varargs`.

4.  **Array:**
    *   PACKAGE: `MODULO_3_API_FONDAMENTALI/ArrayVettori` (studia solo la sottocartella `Array`)
    *   Impara a usare la struttura dati più fondamentale di Java: gli array (sia 1D che 2D) e le `ArrayList`.

5.  **Input dell'Utente:**
    *   PACKAGE: `MODULO_1_BASI_E_CONTROLLO/ScannerInput`
    *   Impara a rendere i tuoi programmi interattivi leggendo l'input dalla tastiera.

MODULO 2: IL CUORE DELLA PROGRAMMAZIONE A OGGETTI (OOP)
==================================================
Obiettivo: Capire come modellare il mondo reale usando gli oggetti.

1.  **Classi e Oggetti (Introduzione):**
    *   PACKAGE: `MODULO_2_OOP/ToString`
    *   Un'introduzione semplice a come si definisce una classe e si crea un oggetto.

2.  **Ereditarietà:**
    *   PACKAGE: `MODULO_2_OOP/Ereditarieta`
    *   Impara come una classe "figlia" può ereditare campi e metodi da una classe "madre".

3.  **Polimorfismo:**
    *   PACKAGE: `MODULO_2_OOP/Polimorfismo`
    *   Scopri come oggetti di tipo diverso possono essere trattati in modo uniforme. È un concetto potentissimo.

4.  **Classi Astratte e Interfacce:**
    *   PACKAGE: `MODULO_2_OOP/ClassiAstratte`
    *   PACKAGE: `MODULO_2_OOP/Interfacce`
    *   Impara a definire "contratti" e "modelli" per le tue classi. Studiali uno dopo l'altro, capendo le differenze.

5.  **Concetti OOP Avanzati:**
    *   PACKAGE: `MODULO_2_OOP/ConcettiAvanzatiOOP`
    *   Approfondisci `Comparable` vs `Comparator`, l'immutabilità e il ciclo di vita degli oggetti.

MODULO 3: LE API FONDAMENTALI DI JAVA
==================================================
Obiettivo: Padroneggiare gli strumenti che Java ti mette a disposizione.

1.  **Stringhe:**
    *   PACKAGE: `MODULO_3_API_FONDAMENTALI/String`

2.  **Date e Orari:**
    *   PACKAGE: `MODULO_3_API_FONDAMENTALI/DataOra`
    *   Impara a usare la moderna e potente API `java.time`.

3.  **Collezioni:**

    *   PACKAGE: `MODULO_3_API_FONDAMENTALI/Iterator`
    *   Impara a usare le strutture dati più comuni: `List`, `Set`, `Map`, `Stack`, `Queue`.

4.  **Generics:**
    *   PACKAGE: `MODULO_3_API_FONDAMENTALI/Generics`
    *   Impara cosa significa `<T>` e perché è fondamentale per la sicurezza delle collezioni.

5.  **Utilità Matematiche:**
    *   PACKAGE: `MODULO_3_API_FONDAMENTALI/Math`
    *   PACKAGE: `MODULO_3_API_FONDAMENTALI/Random`
    *   Scopri le funzioni matematiche e di generazione casuale pronte all'uso.

MODULO 4: PROGRAMMAZIONE FUNZIONALE E MODERNA
==================================================
Obiettivo: Imparare il nuovo paradigma di programmazione introdotto in Java 8.

1.  **Classi Anonime:**
    *   PACKAGE: `MODULO_4_PROGRAMMAZIONE_FUNZIONALE_E_MODERNA/AnonymusClass`
    *   Il "vecchio" modo di creare oggetti al volo. È utile per capire da dove vengono le lambda.

2.  **Espressioni Lambda:**
    *   PACKAGE: `MODULO_4_PROGRAMMAZIONE_FUNZIONALE_E_MODERNA/Lambda`
    *   Impara la sintassi `->` che ha rivoluzionato il modo di scrivere codice in Java.

3.  **Stream API:**
    *   PACKAGE: `MODULO_4_PROGRAMMAZIONE_FUNZIONALE_E_MODERNA/StreamAPI`
    *   Impara a processare collezioni di dati in modo dichiarativo ed elegante. Questo è un argomento ENORME e fondamentale.

4.  **Optional:**
    *   PACKAGE: `MODULO_4_PROGRAMMAZIONE_FUNZIONALE_E_MODERNA/OptionalExample`
    *   Impara il modo corretto e moderno per gestire l'assenza di un valore ed evitare i `NullPointerException`.

MODULO 5: INTERAGIRE CON IL MONDO ESTERNO (I/O)
==================================================
Obiettivo: Imparare a leggere e scrivere dati e a gestire gli errori.

1.  **Gestione delle Eccezioni:**
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/EccezioniTryCatch`
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/EccezioniAvanzate`
    *   Impara a gestire gli errori in modo robusto con `try-catch-finally` e `try-with-resources`.

2.  **I/O su File e Byte:**
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/CreateReadFile` (I/O classico)
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/FileBinari` (I/O binario con DataStream)
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/GestioneByteStream` (Conversione da/a byte[])
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/NIO2` (I/O moderno, il più importante da padroneggiare)

3.  **Database (JDBC):**
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/JDBC`
    *   Impara le basi per connetterti a un database, eseguire query SQL e leggere i risultati.

4.  **Localizzazione:**
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/Localizzazione`
    *   Impara ad adattare la tua applicazione a diverse lingue e regioni.

5.  **Esempi Pratici di I/O:**
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/AudioPlayer`
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/NetworkingSockets` (Bonus, non per l'esame)
    *   PACKAGE: `MODULO_5_INTERAGIRE_CON_IL_MONDO_ESTERNO/InfoDiRete` (Bonus, non per l'esame)
    *   Rivedi questi esempi per vedere applicazioni concrete delle API di I/O.
