RIEPILOGO SULLE API DI UTILITÀ E LIBRERIE DI TERZE PARTI

Nella programmazione reale, un principio fondamentale è **"non reinventare la ruota"**. Per compiti comuni e problemi già risolti, esistono librerie di terze parti robuste, testate e ottimizzate che possono far risparmiare tempo e ridurre i bug.

Questo file riassume tre delle librerie di utilità più famose e utili nell'ecosistema Java.

**NOTA IMPORTANTE:** Il codice nel file `MainAPIUtili.java` è **puramente dimostrativo e commentato**. Per usarlo, sarebbe necessario scaricare i file JAR di queste librerie e aggiungerli al "classpath" del progetto.

--- 1. PROJECT LOMBOK ---

- **Scopo:** Eliminare il codice "boilerplate" (ripetitivo e standard) dalle classi di dati.
- **Come funziona:** Fornisce una serie di **annotazioni** che, durante la compilazione, "generano" automaticamente il codice Java corrispondente.
- **Annotazioni Principali:**
  - **`@Getter` / `@Setter`**: Genera i metodi getter e setter per i campi di una classe.
  - **`@NoArgsConstructor`**: Genera un costruttore senza argomenti.
  - **`@AllArgsConstructor`**: Genera un costruttore con un parametro per ogni campo della classe.
  - **`@ToString`**, **`@EqualsAndHashCode`**: Generano le implementazioni standard di questi metodi.
  - **`@Data`**: Un'annotazione "ombrello" che include `@Getter`, `@Setter`, `@RequiredArgsConstructor`, `@ToString` e `@EqualsAndHashCode`. Trasforma una semplice classe di campi in un POJO (Plain Old Java Object) completo.
  - **`@Builder`**: Implementa il "Builder Pattern", un modo elegante per costruire oggetti complessi.

- **Vantaggio:** Rende le classi di dati estremamente pulite e leggibili, concentrandosi solo sui campi e non sul codice ripetitivo.

--- 2. APACHE COMMONS ---

Apache Commons è una collezione di librerie riutilizzabili. La più famosa è **Commons Lang**.

- **Scopo:** Arricchire il JDK con funzionalità mancanti per la manipolazione dei tipi di base di Java.
- **Classi Principali di `commons-lang3`:**
  - **`StringUtils`**: Una classe enorme con centinaia di metodi per operare sulle stringhe in modo sicuro (gestendo i `null`).
    - `isBlank(String str)`: Controlla se una stringa è `null`, vuota o contiene solo spazi.
    - `isNumeric(String str)`: Controlla se una stringa contiene solo cifre.
    - `defaultIfBlank(String str, String defaultStr)`: Restituisce una stringa di default se quella di input è "blank".
    - `join(Iterable<?> iterable, String separator)`: Unisce gli elementi di una collezione in una stringa, separati da un delimitatore.
  - **`ObjectUtils`**: Fornisce metodi per operare sugli oggetti in modo null-safe.
  - **`ArrayUtils`**: Aggiunge funzionalità mancanti per la manipolazione degli array.

- **Vantaggio:** Fornisce implementazioni robuste e testate per operazioni che altrimenti dovresti scrivere a mano, gestendo correttamente i casi limite come i `null`.

--- 3. GOOGLE GUAVA ---

Guava è una libreria open-source di Google che contiene molte delle utility fondamentali usate internamente dall'azienda. È estremamente vasta e potente.

- **Scopo:** Fornire componenti fondamentali per lo sviluppo di applicazioni Java, con un focus su collezioni, caching, concorrenza e validazione.
- **Funzionalità Principali:**
  - **Collezioni Immutabili:** Fornisce un sistema ricco e potente per creare collezioni immutabili (`ImmutableList`, `ImmutableSet`, `ImmutableMap`). Sono più flessibili e sicure di quelle standard di Java (`List.of`).
  - **Nuovi Tipi di Collezioni:** Introduce strutture dati non presenti nel JDK, come `Multiset` (un `Set` che può contare le occorrenze di un elemento), `Multimap` (una `Map` che può avere più valori per una singola chiave) e `BiMap` (una `Map` in cui anche i valori sono unici e si può cercare per valore).
  - **`Preconditions`**: Una classe per la validazione degli argomenti dei metodi. Permette di scrivere controlli di validità in modo conciso e standardizzato, lanciando eccezioni chiare in caso di fallimento.
    `Preconditions.checkArgument(eta >= 0, "L'età non può essere negativa");`
    `Preconditions.checkNotNull(utente, "L'utente non può essere nullo");`
  - **Utility per Stringhe e I/O:** Fornisce alternative e potenziamenti a quelle del JDK.

- **Vantaggio:** Guava promuove pratiche di programmazione difensiva e fornisce strutture dati potenti che possono semplificare notevolmente problemi complessi.
